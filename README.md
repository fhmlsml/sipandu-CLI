
# SiPandu CLI

CLI Tools to interact with SiPandu (pre-release version)


## Installation

Requirements : python 3.8+ and git

```bash
  pip install git+https://github.com/fhmlsml/sipandu-CLI@{version}
```
    
## Usage/Examples

```bash
$ sipandu --help
$ sipandu login
$ sipandu task create --title "some task"

```
